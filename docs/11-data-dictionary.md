# Словник даних (Data Dictionary)

## Правила маскування PII (Privacy-First)
Дані клієнтів (email, телефон, імена) маскуються на рівні логування додатку (напр., `ki***@gmail.com`, `+380******67`). В базі даних дублювання PII категорично заборонено (нормалізація). Будь-який аудит дій (наприклад, в історії статусів) зберігає виключно `uuid` користувача.
