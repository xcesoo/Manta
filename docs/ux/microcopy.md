# Словник мікротекстів (Microcopy)

## 1. Лейбли та підказки (Labels & Help texts)
| Поле | Label                    | Placeholder / Підказка       |
| :--- |:-------------------------|:-----------------------------|
| `delivery_point_id` | Відділення призначення   | Оберіть відділення зі списку |
| `weight` | Вага (кг)                | Наприклад: 1.5               |
| `amount_due` | Оголошена вартість (ГРН) | 0.00                         |
| `recipient_name` | ПІБ отримувача           | Іванов Іван Іванович         |
| `recipient_phone` | Телефон отримувача       | +XXXXXXXXXXXX                |
| `recipient_email` | Email отримувача         | example@domain.com           |
| `parcel_id` | UUID посилки             | Відскануйте або введіть UUID |

## 2. Повідомлення про помилки (Error Messages)
| Машинний код / Умова | Текст для UI (Людинорозумілий)                                                 |
| :--- |:-------------------------------------------------------------------------------|
| `required` | Заповніть це поле.                                                             |
| `weight_invalid` | Вага має бути більше нуля.                                                     |
| `phone_format` | Введіть телефон у форматі + та 12 цифр.                                        |
| `email_format` | Введіть коректний email.                                                       |
| **409 Conflict** | Відмова: Ліміт відділення (3 посилки) перевищено. Звільніть місце для прийому. |
| **404 Not Found** | Вказану посилку або відділення не знайдено в системі.                          |
| **401 Unauthorized** | Час сесії вичерпано. Увійдіть у систему знову.                                 |

## 3. Тексти станів (State Texts)
* **Loading:** Обробка запиту... (Не перезавантажуйте сторінку)
* **Success (Створення):** Відправлення успішно зареєстровано.
* **Success (Прийом):** Посилку успішно прийнято на відділенні.
* **Empty (Список):** Немає даних для відображення.
